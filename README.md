# student-performance-analysis
This data analysis project utilizes Python, Pandas, and Matplotlib to conduct an Exploratory Data Analysis (EDA) on a student academic dataset (student-mat.csv). The core objective is to evaluate data quality, extract descriptive statistics, identify behavioral factors influencing academic success, and visualize the distribution of final marks.

## 📌 Project Overview
This project is an Exploratory Data Analysis (EDA) of the Student Performance dataset using Python. The objective is to analyze students' academic performance, identify patterns, and visualize important insights.

## 🎯 Objectives
- Load and inspect the dataset.
- Check for missing values and duplicate records.
- Display dataset shape and data types.
- Calculate the average final grade.
- Count the number of students scoring above 15.
- Analyze the correlation between study time and final grades.
- Compare the average performance of male and female students.
- Visualize the data using charts.

## 🛠️ Technologies Used
- Python
- Pandas

- Jupyter Notebook

## 📂 Project Files
- `analysis.py` – Python script containing the complete analysis.

## 📊 Visualizations
The project includes the following visualizations:
- Histogram of Final Grades
- Scatter Plot: Study Time vs Final Grade
- Bar Chart: Average Final Grade by Gender

## 📈 Key Findings
- Calculated the average final grade of students.
- Identified the number of students scoring above 15.
- Analyzed the relationship between study time and academic performance using correlation.
- Compared the average performance of male and female students.

## 🚀 How to Run the Project
1. Download or clone this repository.
2. Ensure `student-mat.csv` is in the same folder as `analysis.py`.
3. Install the required libraries:
   ```
   pip install pandas matplotlib
   ```
4. Run the Python script:
   ```
   python analysis.py
   ```
